## Troubleshooting Guide

### Common Issues & Solutions

**Discussion not appearing**:
- Check category selection
- Verify not marked as spam
- Refresh browser
- Check GitHub status

**Can't create category**:
- Verify admin permissions
- Check Discussions enabled
- Try incognito mode
- Clear browser cache

**Email not received**:
- Check notification settings
- Check spam folder
- Verify email on GitHub
- Add GitHub email whitelist

**Discussion won't lock**:
- Verify admin permissions
- Check discussion isn't already locked
- Try again in 30 seconds
- Check GitHub status page
